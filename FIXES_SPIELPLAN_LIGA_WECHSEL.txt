PROBLEM 1: Kein Spielplan sichtbar
==================================
URSACHE:
  - Die `updateSchedule()` Methode hat immer die erste Liga (schedules.get(0)) verwendet
  - Dadurch haben nur die erste Liga einen Spielplan bekommen
  - Alle anderen 6 Ligen hatten keinen Spielplan

LÖSUNG - RepositoryService.java:
  ✅ updateSchedule() Methode überarbeitet:
    - Findet jetzt den Schedule für die SPEZIFISCHE Liga
    - Nicht mehr schedules.get(0) (falsches Alias)
    - Nutzt league.getId() für Lookup
  
  Vorher:
    Schedule schedule = schedules.get(0); // FALSCH - nur erste Liga
  
  Nachher:
    Schedule schedule = null;
    for (Schedule s : allSchedules) {
      if (s.getLeagueId().equals(league.getId())) {  // RICHTIG - Spezifische Liga
        schedule = s;
        break;
      }
    }

RESULT: 
  ✅ Alle 7 Ligen haben jetzt ihre eigenen Spielpläne
  ✅ RoundRobin wird pro Liga generiert
