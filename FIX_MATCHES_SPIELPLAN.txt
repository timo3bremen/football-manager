FIX: SPIELPLAN & MATCHES FEHLTEN - GELÖST! 🎉
==============================================

PROBLEM:
--------
1. Spielplan-Seite war komplett leer
2. MATCHES Tabelle in der DB war komplett leer
3. Nur Matchdays wurden erstellt, aber OHNE Matches

ROOT CAUSE:
-----------
In `createLeagueWithCPUTeams()` wurde:
  ✅ createSchedule(league)  aufgerufen (erstellt Matchdays)
  ❌ ABER updateSchedule(league) war NICHT aufgerufen (generiert Matches)

Das ist das Kernproblem! Die Matches wurden NIEMALS generiert!

LÖSUNG:
-------

1. RepositoryService.java - createLeagueWithCPUTeams():
   ✅ Hinzugefügt: updateSchedule(league) nach createSchedule(league)
   
   Vorher:
     createSchedule(league);
     System.out.println(...)
   
   Nachher:
     createSchedule(league);
     updateSchedule(league);  // ← Neu! Generiert die Matches
     System.out.println(...)

2. Matchday.java - leagueId Getter/Setter:
   ✅ getLeagueId() hinzugefügt
   ✅ setLeagueId() hinzugefügt

3. RepositoryService.java - createSchedule():
   ✅ Matchday.setLeagueId(league.getId()) für alle Matchdays
   
   Vorher:
     Matchday matchday = new Matchday(schedule.getId(), dayNumber);
     matchdayRepository.save(matchday);
   
   Nachher:
     Matchday matchday = new Matchday(schedule.getId(), dayNumber);
     matchday.setLeagueId(league.getId());  // ← Neu!
     matchdayRepository.save(matchday);

-------
✅ createSchedule() erstellt Matchdays
✅ updateSchedule() generiert Matches (RoundRobin)
